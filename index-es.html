Habilidad

Mi foco de atención esta en crear experiencias web y diseños que funcionan.

My experience on coding over the last few years has let me develop the most useful skills to create a succesful website. You can see them here on the left, and touch on them to understand why i learn them.

  • Front-End

    El Front-End es crucial para hacer websites agradables al ojo del usuario. Con esté puedo esquematizar como la información es estructurada y en que forma interactúa con el consumidor.

  • Base de Datos

    Las Bases de Datos trabajan en el website para almacenar toda la informacion que el usuario nos da. Estos detalles son cruciales si queremos entender cómo el consumidor se comporta.

  • Back-End

    Back-End es el puente entre el Front-End y la Base de Datos. Necesitamos la información que nos da el usuario a través del diseño, y con esta habilidad lo transporto a la Base de Datos.

  • Colocación de la Web

    Esta habilidad comprende como tu página web esta contenida en el internet y la accesibilidad que tiene el usuario para poder alcanzarte.
